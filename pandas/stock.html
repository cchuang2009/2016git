Data Input/Output¶

Data, $X$, look like

$$ \text{Data: } X({\omega})=x \to \text{Python: } X\text{['}{\omega}\text{']}=x$$

where $\omega$ is the state's variable which we observate, (e.g. time, property), and $x$ is its value.

Where the data come from¶

  • From Python
  • From Yahoo finance
  • From CSV files

1. From Python¶

  • pandas.Series([$x_0,\cdots,x_n$], index=['$\omega_0$',$\cdots$,'$\omega_n$']) create a time series data with

    $X[$'$\omega_i$'$]=x_i$

    where rhe variable $\omega_i$ might be datetime, states, ete; here, the datetime is used.

Common Financial Computations¶

  • Moving Average: pandas.rolling_mean(data, denumerator of average) $$ $$ $$ \text{MAvg}=\frac{X_t+\cdots+X_{t+n-1}}{n\text{ days}}$$
  • Returns: data.pct_change( ) which is defined as follows: $$r_t = \frac{p_t - p_{t-1}}{p_{t-1}} = \frac{p_t}{p_{t-1}} - 1$$
